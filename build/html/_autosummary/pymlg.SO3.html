static from_quat(q, order='wxyz')¶

Returns the DCM corresponding to the quaternion representation q.

\[\mathbf{C} = (1 - 2 \mathbf{\epsilon}^T \mathbf{\epsilon}) \mathbf{1} \ + 2 \mathbf{\epsilon \epsilon}^T + 2 \eta \mathbf{\epsilon}^{\times}\]

Note that the final term is positive to abide by robotics convention. This differs from Barfoot (2019).

Parameters:

  • q (list[float] or ndarray of size 4) – quaternion

  • order (str, optional) – quaternion element order “xyzw” or “wxyz”, by default “wxyz”

Returns:

DCM corresponding to q

Return type:

ndarray with shape (3,3)

Raises:

  • ValueError – if q is not of size 4

  • ValueError – if order is not “xyzw” or “wxyz”
